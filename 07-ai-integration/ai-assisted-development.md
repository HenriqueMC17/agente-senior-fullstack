# Desenvolvimento Assistido por Inteligência Artificial (AIA)

## 📌 Contexto

A parceria iterativa entre o Agente autônomo e o Engenheiro base. O processo abandona a figura do "Mago e o Servo cego de Copilot de Aba", passando a tratar IA e o Engenheiro Humano como Pares em Mob Programming Extremo nas janelas das IDEs (Cursor/Windsurf) e nas orquestrações do OS Terminal.

---

## 💻 1. O Pair Programming em Mob-Cognitivo Reduzido (TDD AI)

- A forma ideal de invocar código sem os defeitos e os débitos técnicos dos agentes locais ou de predição é **Gerar Casos e Redigir Assinaturas Prévias** e jamais mandar "Cria tela tal". Dev cria as Props do React Component em Comments, as Rules do TypeScript interface limpas, o teste jest da função isolado no file de Teste da Feature vazia, e envia pro Agente Substituto/Copilot "Faz este Suite específico ficar Válido sem quebrar Complexidade O(n) e passar Global Lints, nada fora da Interface".
- O Modelo AI irá puramente resolver O Foco Isolado sem delirar arquiteturas extras não atreladas e alucinatórias na codebase limpa da feature, preenchendo as Units com excelência sintática extrema da LLM engine base.

## 🔄 2. A Ferramenta Auto-Healing Code Refactor via Agente Bash

Scripts integrados (ex. `.agent/scripts/lint_runner.py --fix-ai`) devem capturar via Logs Bash OS Sub-Process Error a falha TypeScript local do Dev "Type mismatch Zod Parser com Entity String em Data", despachar o contexto da interface para o Local GPT-Agent / Claude Server com as rules carregadas do `workspace-rules/` e aplicar Diff file automático da sugestão limpa na engine (Padrão de Autocorreção Baseado em CLI LLMs).

## 🛡️ 3. A Responsabilidade do Revisão Defensiva Humano-AI Final

- Código originado da inferência AI ainda requer _Due Diligence_ estrito. Arquitetos utilizam a IA paralela na camada das PRs. O Bot Sub-Agent varre e comenta se o dev não validou Boundarys de Security na camada global dos repos. E o Dev sênior faz a revisão das lógicas criadas pelo Agente Builder inicial por vazamentos lógicos e de domínios business (Reviewer-Role AI-Driven `code-review.md`). Ninguém sobe na Main cometa alucinatório de libs de dependência NPM desconhecidas injetada magicamente numa tarde para resolver 1 tooltip de DOM.
